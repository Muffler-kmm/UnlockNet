#  UnlockNet — Система управления подписками

> Двухчастный продукт: пользовательская платежная страница и админ-панель для управления подписками.

##  О проекте

UnlockNet — это полноценная система для управления корпоративными подписками на защищённые каналы связи. Проект состоит из двух частей:

1. **Платежная страница** — для пользователей (выбор тарифа, оплата)
2. **Админ-панель** — для управления (пользователи, платежи, графики, серверы)

Все данные хранятся в `localStorage`, что позволяет использовать проект без бэкенда.

---

##  Функционал

### Пользовательская часть
| Модуль | Описание |
|--------|----------|
| **Выбор тарифа** | LIMIT (200 ₽/мес) или UNLIMIT (300 ₽/мес) |
| **Настройка параметров** | Слайдеры для выбора количества устройств (1–6) и месяцев (1–12) |
| **Дополнительная услуга** | Сброс лимита трафика (+100 ₽) — доступна только для тарифа LIMIT |
| **Авторасчет цены** | Итоговая сумма обновляется в реальном времени |
| **Оплата по СБП** | Имитация оплаты с переходом в платёжную систему |
| **Интеграция с Telegram** | Стилизация под Telegram WebApp, кнопка оплаты в шапке |

### Админ-панель

> Админ-панель не работает в целях защиты, так как данный сайт будет управлять моим проектом, который в дальнейшем будет доработан и выпущен.

| Модуль | Описание |
|--------|----------|
| **Авторизация** | Вход по паролю (хеширование) |
| **Дашборд** | Статистика: всего пользователей, активных подписок, доход сегодня/месяц/всего |
| **Графики** | Визуализация дохода по дням и активных пользователей (Chart.js) |
| **Управление пользователями** | Поиск, фильтры по статусу и тарифу, редактирование (статус, тариф, устройства, продление подписки), удаление |
| **Мониторинг серверов** | Отображение статуса серверов (онлайн/офлайн) |
| **История платежей** | Фильтры по статусу и периоду, общая сумма успешных платежей |
| **Топы пользователей** | Рейтинги по доходу, трафику, приглашениям, устройствам |

##  Стек технологий

| Технология | Что используется |
|------------|------------------|
| **HTML5** | Семантическая разметка, формы |
| **CSS3** | Тёмная тема, адаптивная верстка, кастомные слайдеры, анимации |
| **JavaScript (ES6+)** | Вся логика: выбор тарифа, расчет цены, управление пользователями, графики |
| **Chart.js** | Визуализация данных в админ-панели |
| **CryptoJS** | Хеширование пароля для входа в админ-панель |
| **Boxicons** | Иконки для интерфейса |
| **localStorage** | Хранение данных о пользователях, платежах и настройках |

##  Как запустить

1. **Скачайте репозиторий**:
   ```bash
   git clone https://github.com/Muffler-kmm/unlocknet.git
   
2. Откройте файл index.html (пользовательская часть) в браузере.

3. Для входа в админ-панель откройте admin.html (или index.html с параметром, если реализовано как SPA).

> Все данные хранятся в localStorage. При первом запуске создается администратор с хешированным паролем.
